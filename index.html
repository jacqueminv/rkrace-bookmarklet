RKrace-bookmarklet

Bookmarklet used to edit a race map on Runkeeper by giving a gpx file.

View the Project on GitHub jacqueminv/rkrace-bookmarklet

Editing a race map, one click away

Being a fan of Runkeeper service since a while I was a bit annoyed by their race map creation workflow. Mainly because you have to provide the race map and especially that it has to be a manual process. This can take minutes if not hours for some races.

The little hacker deep inside me kept urging me to have a try fixing this. Here comes my bookmarklet.
